JQUERY

A mais famosa biblioteca JavaScript do mercado fornece uma variação dessa linguagem com uma sintaxe mais amigável, o que simplifica a criação de aplicações. Graças ao jQuery, é possível escrever programas em JavaScript mais facilmente, pois a sintaxe original do JavaScript não é tão fácil de aprender. Com o lema "Write less, do more." (escreva menos, faça mais), jQuery revolucionou o desenvolvimento web, sendo presente em inúmeros projetos atualmente.

NODE.JS

Criado em 2009, o Node.js é um conjunto de ferramentas opensource que permite criar servidores web para execução remota de aplicações JavaScript. Serviços importantes como PayPal, LinkedIn e Groupon usam Node.js para funcionar. Apesar de recente, o Node.js já é utilizado por grandes empresas no mercado de tecnologia, como Netflix, Uber e LinkedIn.
